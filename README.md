# دفتر العملاء (Android Native)

تطبيق إدارة دفتر العملاء يعمل بدون إنترنت باستخدام:
- Kotlin
- Jetpack Compose Material 3
- Room
- MVVM + Repository
- Navigation Compose

## التشغيل
1. افتح المشروع في Android Studio Iguana أو أحدث.
2. مزامنة Gradle.
3. شغّل التطبيق على محاكي أو جهاز Android 8+

## الميزات
- إدارة العملاء والبنود والدفعات
- إجماليات فورية (إجمالي، مدفوع، متبقي)
- بحث وفرز وفلاتر بالتاريخ
- نسخ احتياطي CSV (تصدير/استيراد) مع SAF
- PDF كشف حساب عميل + مشاركة عبر FileProvider
- PDF ملخص لكل العملاء
- دعم RTL وواجهة عربية
